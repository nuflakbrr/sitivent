# Certificate Module - SITIVENT

Certificate hanya tersedia apabila:

```text
certificateEnabled = true
```

dan

```text
Registration Status = CHECKED_IN
```

Certificate terdiri dari:

- Template
- Generated PDF
- Download URL
- Download Time

## Certificate Rules

Certificate dibuat setelah event selesai.

Peserta yang tidak check-in tidak memperoleh sertifikat.

Certificate dapat diunduh melalui:

- Dashboard
- Email
