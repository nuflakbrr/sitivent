# Project Overview - SITIVENT

## Description

SITIVENT adalah sistem informasi manajemen event yang memungkinkan penyelenggara membuat dan mengelola event, sedangkan peserta dapat mendaftar, melakukan pembayaran, melakukan check-in menggunakan QR Code, dan memperoleh sertifikat setelah mengikuti event.

Sistem dirancang untuk mendukung:

- Seminar
- Workshop
- Webinar
- Pelatihan
- Bootcamp
- Conference
- Kompetisi
- Gathering
- Hybrid Event

## Final Notes

SITIVENT merupakan sistem Event Management yang dibangun menggunakan Next.js 16, React 19, Prisma ORM, PostgreSQL, Better Auth, dan TanStack Query.
