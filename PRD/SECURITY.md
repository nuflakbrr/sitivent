# Security Rules - SITIVENT

Seluruh endpoint wajib:

- Authorization
- Authentication
- Validation

Seluruh input wajib:

- Sanitization
- Validation

Gunakan:

- Zod
- Prisma
- Better Auth

Jangan membuat validasi manual apabila sudah tersedia.

Seluruh query database wajib menggunakan Prisma.

Dilarang menggunakan raw SQL kecuali benar-benar diperlukan.

Password tidak boleh disimpan atau diproses secara manual.

Gunakan Better Auth sepenuhnya.
