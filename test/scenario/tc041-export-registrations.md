# Test Case: Export Registrations (TC041)

## Skenario Pengujian
Memastikan Admin dapat mengekspor data pendaftaran peserta event ke format Excel/CSV.

## Prasyarat
- Admin ter-login dan berada di `/admin/transactions/registrations`.

## Kondisi
- **Input**: Klik tombol "Export Data / Excel".
- **Output yang Diharapkan**:
  - Unduhan berkas laporan pendaftaran dimulai.

## Langkah-Langkah Pengujian
1. Buka `/admin/transactions/registrations`.
2. Klik tombol "Export".
3. Verifikasi pengunduhan berkas laporan.
