# Test Case: Free Event Instant Confirmation (TC065)

## Skenario Pengujian
Memastikan pendaftaran event gratis langsung mendapatkan status konfirmasi tanpa perlu verifikasi pembayaran.

## Prasyarat
- Event gratis (harga = 0).

## Kondisi
- **Input**: Klik "Daftar Event".
- **Output yang Diharapkan**:
  - Status pendaftaran langsung `REGISTERED` / `PAID`.

## Langkah-Langkah Pengujian
1. Daftar event gratis.
2. Verifikasi status pendaftaran langsung aktif.
