# Test Case: Name Label Badge Generation (TC067)

## Skenario Pengujian
Memastikan Admin dapat mencetak label nama (name tag/badge) peserta untuk event luring (offline).

## Prasyarat
- Admin di `/admin/transactions/registrations`.

## Kondisi
- **Input**: Klik "Cetak Label Nama".
- **Output yang Diharapkan**:
  - Halaman cetak id card/badge nama peserta terbuka.

## Langkah-Langkah Pengujian
1. Buka daftar pendaftaran.
2. Klik tombol cetak label nama.
3. Verifikasi tampilan cetak badge.
