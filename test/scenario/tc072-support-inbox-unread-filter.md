# Test Case: Support Inbox Unread Filter (TC072)

## Skenario Pengujian
Memastikan Admin dapat menyaring pesan bantuan pelanggan yang belum dibaca (UNREAD).

## Prasyarat
- Admin di `/admin/support/inbox`.

## Kondisi
- **Input**: Klik filter "Belum Dibaca".
- **Output yang Diharapkan**:
  - Hanya menampilkan pesan dengan status `UNREAD`.

## Langkah-Langkah Pengujian
1. Buka kotak masuk bantuan.
2. Filter pesan belum dibaca.
3. Verifikasi daftar pesan terfilter.
